# 🎓 CGPA-Calculator-CPP

A simple C++ program to calculate GPA/CGPA based on course grades and credit hours.

---

## ✨ Features

- Input number of courses
- Enter grade (A/B/C/D/F)
- Enter credit hours
- Calculate grade points
- Compute GPA / CGPA
- Display course-wise summary

---

## 🛠️ Technologies Used

- C++
- Loops
- Arrays (vector)
- Functions
- Conditional statements

---

## 📊 Grade System

A = 10  
B = 8  
C = 6  
D = 4  
F = 0  

---

## 📖 How It Works

1. Enter number of courses  
2. Input grade and credit hours  
3. Program calculates grade points  
4. Computes GPA / CGPA  
5. Displays final result  

---

## 📸 Sample Input

```
Number of courses: 3
Grades: A B C
Credits: 4 3 2
```

---

## 📊 Output

```
GPA / CGPA: 9.11
```
